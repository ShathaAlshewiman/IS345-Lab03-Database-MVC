# Lab 03 – Working with Database Using ASP.NET MVC
# Description
This lab is part of the IS345 – Web Application Development 2 course and was completed for educational and training purposes. The main objective of this lab is to understand how databases are used in web application development using ASP.NET MVC and a PostgreSQL database.

The lab includes creating database tables, inserting and retrieving data, handling database connection errors, and implementing CRUD operations. It also demonstrates using the MVC pattern to connect models, views, and controllers with a relational database.

All data, database records, and examples used in this lab are fictional and are included only for practicing the learned concepts.
# Technologies Used :
ASP.NET MVC

C#

PostgreSQL

SQL

Razor Views
# Learning Outcomes
Connecting ASP.NET MVC applications to PostgreSQL

Performing insert, edit, and retrieve operations

Handling database connection and schema errors

Applying the MVC pattern with a relational database
